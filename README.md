# Quiz Game

## Description
A web-based Quiz Game with user and admin profiles built using PHP, MySQL, and JavaScript.  
Users can play quizzes, track scores, and admins can manage questions and users.

## Features
- User registration and login system
- Admin panel to add/edit/delete questions
- Responsive design for desktop and mobile
- Score tracking for users
- Simple and user-friendly interface
